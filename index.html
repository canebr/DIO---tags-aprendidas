Seções head e body

Voltar ao topo

head e body são as duas seções básicas do HTML.

Na head ficam informações de configuração que não são apresentadas na página que vai ser apresentada para pessoa que vê a página, ela é usada apenas pelo navegador

Na body é onde ficam as informações que serão vistas na página

tags para cabeçalhos e paragrafos

Voltar ao topo

h1 a h6, hr e p são as tags utilizadas para os formatos básicos do texto, definir se a linha é um Titulo ou um paragrafo.

As h1 a h6 definem que a linha é um titulo. Podemos ter 6 tamanhos diferentes de titulos, o tamanho decresce (titulos secundários, terciários, etc) conforme o número aumenta, então h1 é o titulo principal, e o maior deles, h2 o titulo secundário e um pouco menor que o anterior, e assim por diante

A hr serve para criar uma linha horizontal de ponta a ponta da tela, separando duas seções do texto, ou um titulo e uma paragrafo

Na p é onde fica o texto que queremos na nossa página. Devemos usar esta tag para cada paragrafo que colocarmos, se tivermos, por exemplo, 3 paragrafos na sequência, iremos utilizar um par de p para cada um dos paragrafos

tags para alteração no formato da exibição do texto

Voltar ao topo

Nós podemos usar tags como recurso para modificar a forma como o texto é exibido, para dar algum destaque a ele ou para algum uso "padrão" de formatação de textos e de caracteres. ALguns exemplos:

  • mark - utilizado como uma caneta marcadora de texto:

    Este texto demonstra o uso da tag mark

  • sub - Quando necessitamos subscrever alguma coisa na frase:

    Este texto apresenta um subscrito

  • sup - Quando necessitamos sobrescrever alguma coisa na frase:

    uma fórmula matemática deve ser escrita da forma correta, como neste exemplo 32 = 9

  • del - Quando desejamos que uma parte do texto continue na página, mas seja visível que ele foi deletado:

    Eu deletei uma parte do texto mas quero que ainda estja visível.

  • font - É usada para alterar alguma característica da fonte, apesar de poder ser usada no html atualmente é mais normal fazer estas mudanças no css:

    Estou brincando com as fontes

  • abbr - Normalmente utilizado para apresentar abreviações, mas onde queremos que o nome completo apareça caso quem esteja visualizando pouse o mouse por um tempo em cima da abreviatura:

    O Sr. João da Silva fez uma apresentação brilhante ontem na ONU

    Note que a abreviatura está sublinhada com um pontilhado para mostrar que ali ha mais alguma coisa oculta. Não é obrigatório que seja utilizada apenas com abreviações, pode ser utilizada sempre que quisermos que um texto apareça quando o mouse for pousado sobre o que está a vista.

  • blockquote - Serve para assinalar uma seção do texto que vem de uma outra "fonte", como uma citação:

    Eu estou escrevendo este texto, e quero a seguir transcrever um texto dito por outra pessoa:

    Foi testado e provado que este método funciona
    E depois eu continuo o meu texto

listas

Voltar ao topo

ol, ul e li são as tags utilizadas para criar tabelas.

A ol cria listas ordenadas, como a que temos nos "tópicos" no inicio da página

A ul cria listas não ordenadas, como a que temos nos "Links de referências" no final da página

A il é obrigada a ser utilizada para identificar cada um dos itens da lista

abertura e fechamento de tags

Voltar ao topo

As tags do html, com algumas exceções necessitam ser abertas e depois fechadas

Todas as tags ficam entre os sinas de menor (<) e de maior (>).

As tags de abertura aparecem simplesmente entre o menor e o maior - < tag >, já a tag de fechamento apresenta como diferença a barra antes do nome da tag - < /tag >. Observe que coloquei um espaço entre o maior e o menor e o nome da tag, ele foi usado apenas para que o navegador percebesse que não era uma tag, este espaço não deve ser usado quando realmente usamos uma tag
